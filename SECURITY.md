# Security Policy

## Supported Versions

Only the latest version of R97 AI is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

We take the security of R97 AI seriously. If you discover a security vulnerability, please follow these steps:

1. **Do not disclose publicly**: Please avoid opening public issues for sensitive security flaws.
2. **Contact the Developer**: Send a detailed report to the developer via GitHub private messaging or the contact methods listed on [Rehan Ahmad's Profile](https://github.com/Ft976).
3. **Include Details**: Describe the vulnerability, the potential impact, and steps to reproduce it.

### What We Investigate
- API Key exposure risks.
- Prompt injection vulnerabilities that bypass safety filters.
- Data leakage in client-side storage.
- Insecure handling of voice streams.

Thank you for helping keep R97 AI secure!
